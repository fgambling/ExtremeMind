Automatic Scoring System

Overview

A full‑stack web application that streamlines exam grading through AI‑assisted answer evaluation, configurable marking rules, and role‑based access control, featuring manual review tools, Excel data integration, and detailed reporting for admins, test developers, and markers.

Core Features
  • Automated Scoring – Intelligent evaluation with flexible marking rules (case, punctuation, spelling, grammar)
  • Role‑Based Access – Dedicated dashboards for Admins, Test Developers, and Markers
  • Manual Review – Flagged answers for human validation and scoring
  • Excel Integration – Import student responses and export scored results
  • Real‑Time Reporting – Live progress tracking and performance analytics
  • Advanced Linguistic Checks – Spelling & grammar analysis for accurate scoring
